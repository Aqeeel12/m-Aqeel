secure-login-project

Built a secure login system in Python with password hashing and security best practices. Identified vulnerabilities like plain text passwords and implemented SHA-1 hashing to protect user credentials.

  • Built a basic login system using Python.
  • Identified security issues like plain text passwords
  • Used SHA-1 hashing to protect user credentials.
  • Reviewed code for weaknesses and made improvements.
  • Learned how to make code safer and more reliable.

This hands-on project strengthened my understanding of real-world security risks in application development and highlighted the importance of secure design, proper credential management, and defensive coding practices in protecting user data.
